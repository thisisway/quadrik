# Quadrik — Voice & Tone Guide

## Principles

**Direct** — No fluff. Get to the point.  
**Positive** — Energetic, encouraging, action-oriented.  
**Sporty** — Uses sports metaphors naturally, not forcedly.  
**Clear** — Zero ambiguity on actions and outcomes.

---

## Player-Facing Language

Use short, action verbs. Favor energy and community.

### CTAs & Actions
```
Bora jogar?
Criar partida
Entrar no torneio
Reservar agora
Ver horários livres
Subir resultado
Confirmar presença
Chamar jogadores
```

### Confirmations & Feedback
```
Você está dentro!
Partida confirmada
Reserva feita!
Resultado enviado
Inscrição confirmada
```

### Empty States
```
Nenhuma partida aberta por aqui ainda. Que tal criar uma?
Seus torneios vão aparecer aqui. Comece se inscrevendo.
Sem reservas agendadas. Bora marcar uma quadra?
```

## General Rules

1. **Never use passive voice** — "A reserva foi feita" → "Reserva confirmada!"
2. **Avoid technical jargon** — "Erro 500" → "Algo deu errado. Tente novamente."
3. **Use you (você)** — Keeps it personal and direct
4. **Exclamation marks** — Only for genuine positive moments (confirmation, success)
5. **Error messages** — Always explain what happened AND what to do next
6. **Labels** — UPPERCASE for status badges and category chips

---

## Error Message Pattern

```
[What happened] + [How to fix it]

Exemplos:
"Esse horário já está reservado. Escolha outro horário."
"Email inválido. Confira e tente novamente."
"Sessão expirada. Faça login novamente."
"Não foi possível processar o pagamento. Verifique os dados do cartão."
```
